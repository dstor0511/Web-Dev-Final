Accessibility

State 3 ways in which your site is accessible.

  1. Simplicity in the language used in the websites ensures that anyone with an average understanding of the language is going to be able to read and understand the content without any issue.

  2. Headings and Correct semantics use correct HTML tags to ensure that people with disabilities that need a virtual reader are not going to have trouble as the HTML is correctly formatted.

  3. Alternative text has been used in images to assist people with visual disabilities.
